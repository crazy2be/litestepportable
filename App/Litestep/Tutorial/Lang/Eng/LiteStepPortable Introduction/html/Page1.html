Welcome to LiteStep (Portable)

This basic tutorial will show you some basics, and answer some common questions. You can cancel at any time, however, the tutorial will be run the next time you start LiteStep portable.

Using this tutorial

Using this tutorial is fairly straightforward. To the left, you can click on the blue bar to see the index of the current tutorial. At the top, you will see 4 "buttons" on the left, and 3 selectors on the right. The 4 buttons are:
This Button brings you to the first page in the tutorial, not the first page you were on.
This Button brings you to the previous page in the tutorial, not the previous page you were on.
This Button takes you to the next page in the tutorial.
This Button cancels the tutorial :(
And the 3 Selectors, from left to right, select:
  • The Current Tutorial
  • The Current Language
  • The Current Theme (BUGGY!)
Well, that should be enough to get you using the tutorial :)

So let's get started! Hit next to continue.
